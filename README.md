# Amp Fit — CFO Runway & Burn Rate Dashboard

Interactive dashboard for monitoring cash flow, burn rate, and scenario planning.

## Features
- Real-time KPI cards (Cash Balance, Burn Rate, Runway, Net CF)
- Scenario planning with interactive sliders
- 12-month cash runway projection
- Expense waterfall breakdown
- Quick-access preset scenarios
- Monthly comparison tables
